# A-prompt-driven-movie-recommender-system
A simple Netflix-like movie recommender built in Google Colab using Python that understands user requests and suggests suitable movies from a dataset.

IntroductionEach student must design a prompt-driven recommender construct that simulates the behaviour of asimplified Netflix system. The construct should understand a user’s text request (e.g., “light romanticcomedies from the 1990s”) and recommend suitable movies from a dataset. The recommender logic canbe implemented using Flask and Python scripts, or Jupyter Notebook.

Students must use prompt engineering to generate key parts of the system (for example: datasetstructuring, filter logic, explanation generation, or interface text).Every step of AI involvement must be documented with screenshots and written explanations.

Instructions
A. Working ConstructDesign and implement a GUI based recommender engine that:
● Accepts user input (text query) and returns relevant movie recommendations.
● Uses a small dataset (minimum 300 entries with title, genre, cast, plot, and year).
● Demonstrates basic logic for filtering or ranking recommendations.
● Displays recommendations with short explanations.
● It can be run as a simple Flask app using a python notebook or script.

B. Prompt Documentation (Evidence of AI Usage)A clear record showing how AI tools were used to build the construct. This includes:
1. All prompts used — exactly as entered into the AI platform(s).
2.  Screenshots showing conversations or results from AI tools.
3.   Explanations describing:
   ○ What each prompt was meant to achieve.
   ○ How the AI output was used in the project.
   ○ Any adjustments or improvements made to the prompts.
4. Final reflections on what worked well or what could be improved in the prompt process.

Expected Output::

A working Netflix-style recommender constructs that produces grounded movie recommendations fromyour dataset. 
A set of well-designed prompts (with screenshots and explanations) showing how the AIassisted in building and refining your system.

Files to Submit:
1. Construct – containing the working Flask app or Jupyter Notebook, data files, and any scripts.
2.  Prompt Documentation – a single PDF file containing:○ Prompts used○ Screenshots○ Explanations and reflections
